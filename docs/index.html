BLE Medical Gateway

ATOM Lite (ESP32) 用 BLE医療機器ゲートウェイ ファームウェア書き込みツール

対応機器

マイコン M5Stack ATOM Lite
体温計 ニプロ NT-100B
血圧計 ニプロ NBP-1BLE

書き込み手順

  1. ATOM LiteをUSBでPCに接続
  2. 下の「Install」ボタンをクリック
  3. シリアルポートを選択（COM○ または /dev/ttyUSB○）
  4. 書き込み完了を待つ（約30秒）

動作要件

  • Chrome, Edge, Opera のいずれか（Web Serial API対応ブラウザ）
  • ATOM Lite用USBドライバ（下記リンクからインストール）

USBドライバ

ATOM Liteが認識されない場合は、以下のドライバをインストールしてください。

FTDI (FT232) FTDI VCP Drivers
CP210x (Silicon Labs) CP210x Drivers
CH340/CH341 CH340 Driver (Windows)
CH340 Driver (Mac)

※ ATOM LiteはFTDIチップを使用しています。通常はOS標準ドライバで認識されます。

注意事項

  • 書き込み中はUSBケーブルを抜かないでください
  • 書き込み後、ATOM Liteは自動的に再起動します
  • LEDが青色に点灯すればBLEスキャン開始

ファームウェアをインストール

お使いのブラウザはWeb Serial APIに対応していません。
Chrome, Edge, または Opera をお使いください。 シリアルポートへのアクセスが許可されていません。

書き込み後の動作確認

  • 青色LED: BLEスキャン中
  • 緑色LED: 機器と接続済み
  • 白色LED: データ受信
  • 赤色LED: エラー

体温計または血圧計で測定すると、シリアルポートにJSON形式でデータが出力されます。
